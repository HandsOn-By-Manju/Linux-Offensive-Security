# Linux Offensive Security

> A hands-on repository documenting my Linux Offensive Security journey through penetration testing labs, reconnaissance, exploitation, privilege escalation, automation, and practical security methodologies.

---

## 📖 About

This repository serves as my personal knowledge base and portfolio for mastering Linux Offensive Security through hands-on practice.

The goal is to build a structured learning path covering:

- Linux Fundamentals
- Network Enumeration
- Web Application Security
- Service Exploitation
- Linux Privilege Escalation
- Container Security
- Cloud Security
- Post Exploitation
- Security Automation
- Penetration Testing Methodology

Every machine, lab, script, and note in this repository is intended to improve practical offensive security skills.

---

## 🎯 Objectives

- Master Linux penetration testing
- Develop a repeatable reconnaissance methodology
- Improve exploitation techniques
- Learn Linux privilege escalation
- Build automation tools for penetration testing
- Create reusable notes and cheat sheets
- Document lessons learned from every lab
